VineBond Figma Element Assets

Files included:
- primary-button.svg
- top-bar-icon-vinemap.svg
- custom-cursor-wineglass.svg
- map-marker-vineyard.svg
- filtration-dropdown.svg
- vinebond-home-ui-kit.svg (all elements on one canvas)

How to add to Figma:
1) Open your Figma file.
2) Drag and drop any SVG file onto the canvas.
3) For the complete kit, import vinebond-home-ui-kit.svg.
4) In Figma, right-click imported groups and create components/variants as needed.
